# Naming Scheme

Naming scheme can be as follows:
```
MovieTitle (Year) {Identifier} /
├── MovieTitle (Year) {Identifier}.mkv
├── MovieTitle (Year).[language code].srt
├── cover.jpg
└── poster.jpg
```

Example:
```
Avatar (2009)/
├── Avatar (2009).mkv
├── Avatar (2009).en.srt
└── cover.jpg
```
